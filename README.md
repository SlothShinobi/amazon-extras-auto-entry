# Amazon Extras Giveaway Auto-Entry

A Tampermonkey script that automates entering giveaways on Amazon's employee benefits portal.

## What is Amazon Extras?

Amazon Extras (extrasforamazon.com) is a benefits platform available to Amazon employees offering discounts, deals, and giveaways from various vendors.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) browser extension
2. [Click here to install the script](https://raw.githubusercontent.com/YOUR_USERNAME/amazon-extras-auto-entry/main/amazon-extras-giveaway-auto-entry.user.js)

## How It Works

The script:
1. Scans the page for eligible giveaways
2. Navigates to each giveaway detail page
3. Clicks the "Enter Giveaway" button
4. Returns to the main page
5. Repeats until all giveaways are processed

## Requirements

- Tampermonkey browser extension
- Access to Amazon Extras (Amazon employees only)
